A WooCommerce plug-in enabling you to create Shiplee shipment labels directly from the WooCommerce Orders section.

== Description ==

[Shiplee](https://shiplee.com/) is an online platform for same day and next day delivery of parcels. This plug-in for WooCommerce enables you to create Shiplee shipment labels directly from the WooCommerce Orders section.

### Getting started
1. [Create a Shiplee account](https://app.shiplee.com/auth/register) in case you don't have one yet.
2. [Create an API key](https://app.shiplee.com/shop/api-keys) and store it in a safe place.
3. In your WordPress admin, click on the WooCommerce nagivation item and then on Settings, go to the 'Shiplee shipping plug-in' section in the Shipping tab, fill in your API key and save it.
4. You will now be able to create Shiplee shipment labels from the WooCommerce Orders pages in the admin!

### How it works
This plug-in adds an 'Add label' link on both the 'Ship to' column in the WooCommerce Orders overview page and on the WooCommerce Orders detail page. You can create a Shiplee shipment label by clicking on one of these links. A dialog will open after clicking on the 'Add label' link and in this dialog you can choose your shipping product. After choosing the product that is right for your shipment, you can fill in the necessary additional data, click on 'Create label' and your label will be created!

== Frequently Asked Questions ==

= Where can I get my API key? =

In case you don't have a Shiplee account yet, create one [here](https://app.shiplee.com/auth/register). On [this page](https://app.shiplee.com/shop/api-keys) you can create your API key.
